Transfer Files Securely Via the Network

Tata cara Transfer files

Posted by azzaid on October 29, 2018

Langkah pertama

Pastikan file dan zsh sudah ada,jika ssh belum ada maka anda bisa menginstallnya dengan mengikuti command dibawah :

[fesa@fesa /]~ sudo apt-get install ssh

Langkah kedua

jika ssh sudah terinstall,maka selanjutnya pastikan user pada file anda bukan user root.kita bisa mengubahnya dengan mengikuti command dibawah :

[fesa@fesa /]~ chown -R (nama user)(nama user) (nama file /folder)

setelah command dijalankan :

Langkah ketiga

sekarang jika semuanya sudah diubah maka sekarang kita bisa mengirim file ke server satu lagi dengan command berikut :

1. kita akan menggunakan scp.scpadalah command line yang hanya bisa untuk transfer satu file dan belum tentu permissionsnya ikut ke ganti.

contoh command linenya :

[fesa@fesa /]~ scp (nama file) (nama user)@(ip server yang ingin dikirimi file):/(nama directory)/(nama directory

2. kita akan menggunakan rsync.rsync adalah command line yang bisa untuk transfer banyak file dan permissionsnya ikut ke ganti atau bisa disebut juga pembackupan.

contoh command linenya :

[fesa@fesa /]~ rsync (nama file) (nama user)@(ip server yang ingin dikirimi file):/(nama directory)/(nama directory
