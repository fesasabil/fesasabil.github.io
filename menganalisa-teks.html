Mengenal Regular Expressions

Fungsi dan penggunaan Regular Expressions

Posted by azzaid on October 18, 2018

Regular Expressions adalah String teks yang menggambarkan pola penelusuran tertentu.

Aplikasi dan bahasa pemrograman yang berbeda menerapkan ekspresi reguler sedikit berbeda. Kami hanya akan menjelajahi bagian kecil dari cara yang grep menggambarkan pola-polanya :

Beberapa kegunaan Regular Expressions :

1. Literal Matches

Contoh di bawah, ketika kami menelusuri kata "licensi", kami sebenarnya mencari ekspresi reguler yang sangat sederhana, yang cocok dengan string karakter yang tepat "licensi".

[root@fesa /]# grep "license" nama file.

2. Untuk mencari karakter atau kata pada awal baris pada sebuah file.

contoh :

[root@fesa /]# grep "^GNU" data.txt

3. Untuk mencari karakter atau kata pada akhir baris pada sebuah file.

contoh :

[root@fesa /]# grep "and$" data.txt

4. Untuk mencari potongan karakter atau kata pada sebuah file.

contoh :

[root@fesa /]# grep "..cept" data.txt

5. Untuk mencari 2 karakter atau 2 kata pada sebuah file.

contoh :

[root@fesa /]# grep "t[wo]o" data.txt

6. Untuk mencari huruf dengan awalan “T” dan dilanjutkan sesuai abjad kecuali huruf “e” di karenakan menggunakan “^”.

contoh :

[root@fesa /]# grep ‘^T[a-z][^e] ’ data.txt

7. Untuk mencari huruf “T” dan diurutkan dari “a-z” dilanjutkan dengan huruf “aiueo”.

contoh :

[root@fesa /]# grep ‘^T[a-z][aiueo] ’ data.txt

8. Untuk mencari huruf “t/T” yang bersambung “he” disebuah file,maka akan keluar semua huruf “the” yang berawalan besar maupun kecil di tengah maupun di awal kalimat sebuah file

[root@fesa /]# grep ‘\<[tT]he\>’ data.txt

9. Untuk mencari huruf yang double pada sebuah file dari huruf “a-z”.

contoh :

[root@fesa /]# grep ‘\([a-z]\)1’ data.txt
