Mendownload aplikasi Whatsapp desktop

Tutorial Mendownload Whatsapp dekstop

Posted by azzaid on October 16, 2018

Mendownload Whatsapp di Windows 8.1+ :

  1. Pada browser komputer Anda, kunjungi halaman unduhan untuk mengunduh file .exe.
  2. Saat unduhan selesai dilakukan, buka WhatsApp.exe untuk memasang WhatsApp pada komputer Anda.
  3. Ketika pemasangan selesai, buka WhatsApp pada komputer Anda dan pindai kode QR untuk masuk.

Mendownload Whatsapp di Mac OSX 10.9+ :

  1. Pada browser komputer Anda, kunjungi halaman unduhan untuk mengunduh file .zip.
  2. Saat unduhan selesai, buka file zip untuk menjalankan aplikasi WhatsApp.
  3. Pada pemasangan pertama Anda, aplikasi akan meminta jika Anda ingin menambahkan WhatsApp ke folder Aplikasi dan dock desktop Anda.  
  4. Ketika pemasangan selesai, buka WhatsApp pada komputer Anda dan pindai kode QR untuk masuk.

Mendownload Whatsapp di Ubuntu :

  1. Pada browser komputer Anda, kunjungi Link
  2. Saat unduhan selesai, buka file zip untuk menjalankan aplikasi WhatsApp.
  3. Pada pemasangan pertama Anda, aplikasi akan meminta jika Anda ingin menambahkan WhatsApp ke folder Aplikasi dan dock desktop Anda.  
  4. Ketika pemasangan selesai, buka WhatsApp pada komputer Anda dan pindai kode QR untuk masuk.

Setelah file di download,silhkan buka terminal kemudian masuk ke directory tempat file tersebut berada,semua file yang di download biasanya akan masuk ke directory~/Downloads 

$ cd Download

kemudian install dengan package manager dpkg.

$ sudo dpkg -i whatsapp-webapp_1.0_all.deb

Tunggu sampai proses install selesai.Apabila sudah terisntall,klik tombol Dash Home kemudian ketik WhatsApp untuk menjalankan aplikasi WhatsApp via desktop.
