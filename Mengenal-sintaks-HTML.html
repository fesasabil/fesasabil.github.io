Mengenal sintaks HTML

Beberapa sintaks HTML

Posted by azzaid on October 18, 2018

1. Tag   <I> </I>

Berfungsi untuk Menyisipkan tulisan miring atau Membuat tulisan menjadi miring.

2. Tag   <u> </u>

Menyisipkan tulisan bergaris bawah atau Membuat tulisan menjadi bergaris bawah.

3. Tag   <b> </b>

Menyisipkan tulisan tebal atau Membuat tulisan menjadi tebal.

4. Tag   <li> </li>

Menyisipkan bullet.

5. Tag   <marquee> </marquee>

Menyisipkan tulisan berjalan . “welcome to website” bergerak dari kiri ke kanan atau dari atas ke bawah.

6. Tag   <blink> </blink>

Menyisipkan tulisan berkedip kedip.

7. Tag   <center> </center>

Menyisipkan tulisan dalam bentuk rata tengah atau Membuat tulisan menjadi rata tengah.

8. Tag   <title> </title>

Menyisipkan title html atau untuk membuat judul HTML.

9. Tag   <head> </head>

Menyisipkan bagian head pada sebuah HTML.

10. Tag   <cite> </cite>

Mendefinisikan kutipan.

11. Tag   <abbr> </abbr>

Digunakan untuk menguraikan satu ungkapan yang disingkat dan Anda dapat memberikan informasi bermanfaat kepada penelusur-penulusur page source(halaman sumber kode) / pembaca layar, sistem terjemahan dan mesin pencari yang berasal dari singkatan yang sudah diuraikan, tetapi saat di browser uraian tersebut tidak akan tampil dan hanya sebagai informasi saja.

12. Tag   <button> </button>

Mendefinisikan sebuah tombol diklik.

13. Tag   <span> </span>

Mendefinisikan sebuah section dalam dokumen.

14. Tag   <strike> </strike>

Untuk memberi garis tengah pada teks/mencoret teks, fungsi tag ini sama dengan tag <del>

15. Tag   <samp> </samp>

Mendefinisikan contoh keluaran dari program computer.
