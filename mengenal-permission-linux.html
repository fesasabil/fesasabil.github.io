Mengenal file Permissions

Fungsi dan penggunaan file Permissions

Posted by azzaid on October 27, 2018

file Permissions adalah sebuah izin untuk mengubah atau mengedit sebuah file.

Fungsinya Untuk mengubah atau mengedit file yang dimiliki oleh root, sudo harus digunakan - silakan lihat RootSudo untuk detailnya

Beberapa kegunaan file Permissions :

1. Permissions Read

Dengan permissions ini kita hanya bisa membaca sebuah file dan tidak bisa untuk mengedit atau mengubah sebuah file.dan permissions ini bernilai 4.

2. Permissions Write

Dengan permissions ini kita tidak hanya bisa membaca akan tetapi kita juga bisa mengedit atau mengubah sebuah file.dan permissions ini bernilai 2.

3. Permissions Execute

Dengan permissions ini kita memiliki izin untuk masuk kepada sebuah directory.dan permissions ini bernilai 1.

Beberapa sintag untuk Permissions :

1. Sintag untuk mengubah semua permission menjadi -rwxrwxrwx.

contoh :

[root@fesa /]# chmod u+rwx data.txt

2. Sintag untuk mengubah permission menjadi -rwx---rwx .

contoh :

[root@fesa /]# Chmod g-rwx data.txt

3. Sintag untuk mengubah permission menjadi -rwx------ .

contoh :

[root@fesa /]# Chmod o-rwx data.txt
