Mengenal command find

Fungsi dan penggunaan command find

Posted by azzaid on October 22, 2018

Fungsi command find :

Untuk mencari letak sebuah karakter atau folder atau file dengan menggunakan terminal di linux

Penggunaan command find :

1. Untuk mencari letak file berdasarkan nama file tersebut dengan perintah :

find / -name namafile.txt

2. Untuk mencari file dengan nama.

[root@fesa /]# find ~ -name "*e*"

[ fesa@fesa-Presario-CQ42-Notebook-PC /]~ find ~ -name "*e*"

3. Untuk mencari file dengan dua karakter dimana karakter pertama bebas dan karakter keduanya “e” maka perintahnya akan seperti ini :

[root@fesa /]# find ~ -name "?e"

[fesa@fesa-Presario-CQ42-Notebook-PC/]~ find ~ -name "?e"

4. Untuk mencari file di suatu folder tertentu, contoh kita akan mencari file pada folder / yang nama filenya diawali dengan ‘C’ maka perintahnya seperti ini :

[root@fesa /]# find ~ -name "c*"

[fesa@fesa-Presario-CQ42-Notebook-PC/]~ find ~ -name "c*"

5. Untuk mencari file berdasarkan tipe tertentu contohnya kita akan mencari direktori-direktori dibawah /usr/local/ maka perintahnya seperti berikut :

[root@fesa /]# find ~ -type d

[fesa@fesa-Presario-CQ42-Notebook-PC/]~ find ~ -type d

6. Untuk mencari tahu berapa jumlah direktori pada /usr/local/ kita tinggal piping perintah tadi ke program wc :

[root@fesa /]# find /usr/local -type d | wc -1

[fesa@fesa-Presario-CQ42-Notebook-PC/]~ find /usr/local -type d | wc -1

7. Untuk mencari file bertipe block special di direktori /dev/ tetapi yang memiliki nama file diawali dengan hda :

[root@fesa /]# find /usr/local -type b -name "hda"

[fesa@fesa-Presario-CQ42-Notebook-PC/]~ find /usr/local -type b -name "hda"

Pada contoh diatas kita mengkombinasikan argumen type dan name dan berarti anda sudah mengerti cara menggunakan find dengan dua argumen. Untuk notasi tipe-tipe file yang dikenal oleh linux akan dijelaskan seperti berikut :

  1. b untuk block special
  2. c untuk character special
  3. d untuk direktori
  4. p untuk fifo
  5. f untuk regular file
  6. l untuk symbolik link
  7. s untuk socket
8. Untuk pencarian pada folder /home atau root yang ukuran filenya lebih dari 200 KB :

[root@fesa /]# find ~ -size +200k

[fesa@fesa-Presario-CQ42-Notebook-PC/]~ find -size +200k

9. Untuk mencari file pada folder bin yang berukuran kurang dari 10 KB dan bertipe regular file :

[root@fesa /]# find /bin/ -size 10k -type f

[fesa@fesa-Presario-CQ42-Notebook-PC/]~ find /bin/ -size 10k -type f

10. Untuk mencari file yang ukurannya diantara atau dalam range tertentu :

[root@fesa /]# find /bin/ -size +20k -size -30k

[fesa@fesa-Presario-CQ42-Notebook-PC/]~ find /bin/ -size +20k -size -30k

Dari tadi kan kita mencari file dengan ukuran Kilobyte gimana mencari file yang satuannya lain, nah ini beberapa contoh argumen yang dapat digunakan :

  1. b untuk blok 512 byte
  2. c untuk byte
  3. w untuk word
  4. k untuk kilobyte (1024)
  5. M untuk mega byte (1048576)
  6. G untuk giga byte ( (1073741824)
11. Untuk mencari file berdasarkan kepemilikan user dari file tersebut menggunakan perintah :

[root@fesa /]# find / -user root -name "*.iso"
