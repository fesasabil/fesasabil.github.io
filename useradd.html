Mengenal, Membuat User dan Group

Pengertian dan fungsi User dan Group

Posted by azzaid on November 7, 2018

Pengertian User :

User adalah komputer yang di hubungkan oleh sistem operasi agar dapat mengerjakan perintah atau instruksi- instruksi yang di berikan oleh user.

Beberapa command untuk memanage User :

1. ADDUSER/USERADD = perintah untuk membuat user account baru yang tidak terdapat pada sistem.

option yang digunakan :

  • -u = nomor identitas user (UID).

  • -g = nomor identitas group (GID).

  • -d = menentukan path home direktori user secara manual.

  • -s = shell yang akan digunakan user.

  • -c = komentar tentang user.

    • 2. PASSWD = perintah yang digunakan untuk mengubah password user yang terdapat pada sistem.

    3. USERMOD = perintah yang digunakan untuk memodifikasi user acount yang ada pada sistem linux.

    4. USERDEL = digunakan untuk menghapus user yang ada pada sistem.

    Beberapa perbedaan useradd dan adduser :

  • useradd = menciptakan user tanpa mengeset password.
  • adduser = menciptakan user dengan melakukan setting password terlebih dahulu.

  • useradd = menciptakan user tanpa ada home dirktorinya.
  • adduser = menciptakan user serta home direktorinya.

  • useradd = menciptakan user tanpa memberikan keterangan lengkap mengenai user tersebut.
  • adduser = menciptakan user dengan memberikan keterangan lengkap si user tersebut.

    • Pengertian Group :

    Group adalah kumpulan dari beberapa user yang menjadi satu kelompok kerja.

    Beberapa command untuk memanage Group :

  • GROUPADD = perintah untuk membuat group baru.

  • GPASSWD = perintah untuk menambahkan user ke dalam group atau menghapus user dari dalam group.

  • GROUPMOD = digunakan untuk memodifikasi group account yang ada di sistem.

  • GROUPDEL = digunakan untuk menghapus suatu group dari sistem.

    • File-file yang berhubungan dengan User dan Group :

  • /etc/passwd = file tempat menyimpan data semua user yang ada di sistem linux.

  • /etc/shadow = file ini berisi data password yang sudah terenkripsi untuk user linux.

  • /etc/group = file ini berisi keterangan tentang group.
